# vue-switch-language
A vue simple application for multilingual website.
- SEO best practices in routing
- Ablility to easily switch beween rtl and ltr
- Ablility to change language of contents

## For running the application
- npm install
- npm run serve

### Dependencies
- vue.js
- vue-router
- vuex
- vue-multiselect
- vue-i18n
- vue-country-flag
